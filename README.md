# Tic Tac Toe Game (Python)

This is a simple Tic Tac Toe game implemented using Python.

## Features
- Two player gameplay
- Basic AI logic
- Command line interface
- Beginner friendly Python project

## How to Run
Clone the repository:
git clone https://github.com/Kanaka-VY/tic-tac-toe-python.git

Run the file:
python tic_tac_toe.py
